# GitHub Upload Checklist

Before publishing the repository:

- [ ] Confirm `.env` is not committed
- [ ] Keep `.env.example` with placeholders only
- [ ] Add screenshots to the README after testing locally
- [ ] Verify the sample PDFs open correctly
- [ ] Run `pytest -q`
- [ ] Run the app at least once with `streamlit run app.py`
- [ ] Update the repository description on GitHub
- [ ] Add topics such as `azure-openai`, `rag`, `pdf-chatbot`, `streamlit`, `dio`, and `dp100`
- [ ] Optionally replace the included sample PDFs with a topic of your own interest
